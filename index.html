Run Unix executable binary on MATLAB

2018-04-18
Software
As a MATLAB guy and *nix guy, you may want to fulfill everything on MATLAB, or on command line. I will choose the former one. Call SYSTEM command in MATLAB MATLAB has its own bssh PATH, which is different from the *nix PATH. But a workaround consists in modifying the script (/Applications/MATLAB_R*****.app/bin/matlab) that launches MATLAB. $ cd /Applications/MATLAB_R*****.app/bin/ $ vi matlab # add the following line after #!/bin/sh $ export PATH:$PATH Then, add this matlab into your ~/.
