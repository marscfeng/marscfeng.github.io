when i upgraded my ubunutu from 16.04 to 16.10, i found that i cannot use qq_intel on this platform. It threw out an error indicating that this qq_intel is too old for my QQ account. After some trials, i found qq_light_deepin and it works well on my ubuntu 16.10 platform.

install crossover

download CROSSOVER 16.0 and install it

$ sudo dpkg -i crossover_16.0.0-1.deb
# in case of lacking of some dependences, try follows to fix the broken 
$ sudo apt-get -f install

download CROSSOVER crack and replace the old one under the installation path

$ sudo cp winewrapper.exe.so /opt/cxoffice/lib/wine

install QQ_light_deepin on corssover

download the published bottles of crossover about QQ_light_deepin and install it

$ sudo cp -R ./data.tar.xz/opt/cxoffice/support/apps.com.qq.im.light /opt/cxoffice/support/

open crossover and qq_light will be there
