下载

CWP/SU

设定 CWPROOT 环境变量

习惯将软件安装在/usr/local下，因此就在～/.bashrc中添加 CWPROOT的路径：


# set env for CWP

export CWPROOT=/usr/local/CWP

export PATH=${CWPROOT}/bin:$PATH

Note： CWPROOT根目录必须要有一般用户的读写权限！因此， 我把/usr/local下子文件全部添加一般用户读写权限，这样整个安装过程都不需要root用户权限！


sudo chmod 777 -R /usr/local

安装前准备

解压安装包得到一个Src的源码文件夹

  1. 检测CWPROOT变量是否设定

./chkroot.sh

  1. 创建安装目录

./mkdirectories.sh

  1. 到./configs目录下寻找符合自己电脑型号的Makefile_config文件替换Src下原始的Makefile_config文件
  1. 个人癖好

把源码文件夹拷贝到CWPROOT目录下

  1. 修改Makefile_config文件

其实基本的安装也不需要修改config文件，这里主要就是X11和Xm.h文件库的问题，我忽略！

安装

在Src下执行


make install   (to install the basic set of codes)

make xtinstall (to install the X-toolkit applications)

make finstall (to install the Fortran codes)

测试

$ suplane |suxwigb &
