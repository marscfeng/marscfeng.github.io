modify citation style in mendeley

2018-03-31
Software

Contents

作为文献管理工具，Mendeley可以说深得民心。当然，国内用户可能会对其不稳定的服务器不爽，但是不算是影响生产力的问题。唯一让我崩溃的一件事是它的文件引用格式。用Latex写文章的时候，这个问题也是不存在，可是boss只用word，问题便来了。近百篇的参考文献，批量化的格式引用势在必行。

好在发现mendeley提供了Citation Style Language (CSL) editor，尽管不是那么好用，但总算编辑好几个常用期刊的CSL。懒得自己耗时间的，请直接Copy相应的link，然后在Mendeley>View>Citation Style>More Styles>Get more styles下载安装。

# GJI
http://csl.mendeley.com/styles/218510201/gji
# Geophysics
http://csl.mendeley.com/styles/218510201/geophysics-fcheng
# Survey in Geophysics
http://csl.mendeley.com/styles/218510201/survey-in-geo-springer-basic-author-date-2
# Elsevier
http://csl.mendeley.com/styles/218510201/elsevier-harvard-3

具体来说，CSL Editor编辑新参考文献格式需注意：

  1. 服务器不稳定，偶尔不能访问、不能保存；
  2. 编辑或修改模版时，首先修改info并保存为自己的格式名；
  3. 编辑Layout时注意每个group及相应的条件判断；
  4. 所有的格式设定，逗号、句号都有相应的语句来控制，请细心添加、查找、修改、保存。
