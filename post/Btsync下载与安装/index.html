1.下载linux平台

Btsync

2. 解压并添加环境变量


# cd /usr/local

# sudo mkdir btsync

# sudo tar xzf stable -C btsync

# sudo rm stable

# sudo gedit ~/.bashrc

> export PATH=$PATH:/usr/local/btsync

3. 配置btsync

btsync安装包中没有配置文件，需要我们创建一个


# btsync --dump-sample-config > btsync.config

# sudo gedit /usr/local/btsync/btsync.config

> "device_name": "Mars Linux",

> "login" : "mars"

> "password" : "1314"

其他的不做修改

4. 通过浏览器进入Btsync同步界面


touch /var/log/btsync.log # 创建日志文件

btsync --config /usr/local/btsync/btsync.config --log /var/log/btsync.log  #启动btsync

127.0.0.1:8888/gui/  # 进入浏览器同步界面

5. 设置一个无效用户给与其btsync开机启动权限


# sudo useradd btsync

# chown -R btsync:root /usr/local/btsync

> 修改开机启动文件

# sudo gedit /etc/rc.local

> (exit 0 之前添加)

sudo -u btsync /usr/local/btsync/btsync --config /usr/local/btsync/btsync.config --log /var/log/btsync.log
