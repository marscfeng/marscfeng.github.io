Thanks HUQI’s share!

check the lic file 'huqi/trial.key' from above link

Take MAC for example:

  1. download beyond compare package
  2. install
  3. replace trial.key under /Applications/Beyond\ Compare.app/Contents/Resources/trial.key with above license
  4. save trial.key & restart app

Usage:

  1. install command line tool
e.g.,
$ bcompare new.f90 old.f90
$ bcompare /Folder1/ /Folder2/
  1. install Beyond Compare plugin on sublime text

Notes:

  1. Refer the comments on huqi’s Githu for more info
  2. This license works for 4.1.x & 4.2.2 & 4.2.3
  3. This license works for MAC/Win/Linux
