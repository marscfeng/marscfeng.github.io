sukeyword 查看su关键字

%Usage:
sukeyword -o  %在终端输出segy.h
sukeyword [str] %在终端输出segy.h中所有与[str]匹配的字段

常见keywords包括tracl，fldr，offset，ns等。

sugethw 获取地震数据某一关键字值

%Usage:
sugethw < stdin key=KEYWORDS
%
$ sugethw < 1.su key=tracl
 tracl=1
 tracl=2
 ...
 tracl=140
 tracl=141

suwind 截取特定窗口地震数据

%Usage
suwind < stdinkey=KEYWORDS min=MIN max=MAX > stdout 截取关键字选定范围数据
suwind < stdinkey=KEYWORDS min=MIN count=NUM > stdout
suwind < stdin tmin=TMIN tmax=TMAX > stdout 截取特定时间窗口数据

suxwigb 用X-windows绘制变面积地震记录波形图

%Usage:
suxwigb < stdin [key=KEYWORDS] %关键词未指定的情况下，xigb将以头文件中的比例坐标定义xy轴
$ suxwigb < 1.su  title=XWIGB-default label1="TIME"  
$ suxwigb < 1.su  title=XWIGB-offset label1="TIME" label2="OFFSET" key=offset
$ suxwigb < 1.su  title=XWIGB-trace label1="TIME" label2="TRACE" key=tracl
$ suxwigb < 1.su  title=XWIGB-X label1="TIME" label2="X-Coordinate" key=gx

-1.标量图因此不适合数字化调整坐标轴长短，直接拖拽窗口来更改坐标轴比例 -2.在图形窗口上按q键退出图形

supsimage 绘制灰度或者彩色矢量地震图

%Usage
supsimage < stdin > stdout 
%
$ supsimage < 1.su title=IMAGE label1="TIME" label2="OFFSET" f2=-1000  hbox=4 wbox=4 > 1.eps
$ supsimage < 1.su title=IMAGE label1="TIME" label2="OFFSET" f2=-1000  hbox=7 wbox=4 wrgb=1.0,0,0 grgb=0,1.0,0 brgb=0,0,1.0 grid1=dash grid2=dash legend=1 lnice=1 > 2.eps
$ suwind < 1.su key=tracl min=40 max=100 | supsimage  title=IMAGE-xwin label1="TIME"   hbox=7 wbox=4 > 3.eps
$ suwind < 1.su tmin=0.8 tmax=1.0 | supsimage title=IMAGE-twin label1="TIME" label2="OFFSET" f2=-1000  hbox=7 wbox=4 > 4.eps

suxcontor 绘制等值线标量地震图

%Usage:
suxcontor < stdin > stdout
%
$ suxcontor < 1.su title=CONTOR label1="TIME" label2="OFFSET" f2=-1000

supsgraph 绘制矢量地震记录曲线

%Usage:
supsgraph < stdin  > stdout
%
$ supsgraph < 1.su x1beg=0.8 x1end=1.0 x2beg=-1.0e-5 x2end=1.0e-5  label1="TIME" > 5.eps
$ supsgraph < 1.su nplot=40 x1beg=0.8 x1end=1.0 x2beg=-1.0e-5 x2end=1.0e-5  label1="TIME" > 6.eps
$ suwind < 1.su key=tracl min=40 max=40 | supsgraph x1beg=0.8 x1end=1.0 x2beg=-1.0e-5 x2end=1.0e-5  label1="TIME" > 7.eps

merge2/4 矢量图合并

%Usage:
merge2 f1 f2 > fn
merge4 f1 f2 f3 f4 > fn
%
$ merge2 6.eps 7.eps > temp.eps

Optional parameters

更多绘图参数请参考xwigb，psimage，psgraph
