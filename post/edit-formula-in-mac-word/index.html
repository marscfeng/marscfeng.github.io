edit formula in mac word

2016-10-18
Software

Contents

  • LatexiT
  • Latex2word-Equation Extension
  • Latex在线编辑器

对于windows用户，给word，ppt添加公式可以直接使用mathtype。 如果你喜欢latex的，mathtype也支持latex语法，所以在microsoft office下面使用mathtype还是很不错的。

对于mac用户，个人认为latex还是很方便的，不过mac office也不错，尤其是当别人只能使用office的时候，你给个latex的文稿别人也不好给你批阅。但是，mathtype支持800多种应用，可不支持mac下面的应用，包括mac下面的office。对于mac下面给word添加公式，我给出以下三种基于latex的解决方案，不喜欢或者不想用latex的可以略过。

LatexiT

LatexiT是mac下面原生态的基于latex的公式编辑器。LatexiT将latex语法编译、渲染成相应的latex公式，支持复制、导出，然后拷贝到word中即可。

Latex2word-Equation Extension

Latex2word-Equation 是Zhang基于JavaScript开发的chrome插件。该插件可以给chrome添加一个右键菜单Latex2word-Equation，可以将复制的latex公式转换成office可以识别的math语法，及word公式。个人感觉还不错。

Latex在线编辑器

在线Latex公式编辑器类似于LatexiT，不过是一个在线编辑器。个人感觉其语法识别比LatexiT要强，就是能包容一些小错误。Latex在线编辑器和LatexiT的渲染效果都不错，导出为pdf格式图片保存到本地，在拷贝到word中就OK了。这一步相比LatexiT多了一个保存到本地的步骤。

#总结

  1. 以上三者，首推LatexiT，因为界面简单，操作也简单，不需要保存图片到本地；

  2. 但是偶尔会因为语法不识别让人很捉急，也许是语法还没搞清楚吧，这个时候可以使用Latex在线编辑器；

  3. 最后如果想要原滋原味的word效果，那就Latex2word-Equation插件吧，但是这个插件转换的word公式偶尔也还是需要局部调整一下的。不过word自己的公式渲染真是不想评价。
